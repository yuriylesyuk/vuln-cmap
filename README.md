# A Conceptual Map of Vulnerability Terminology (vuln-cmap)

An interactive, web-based visualization of vulnerability frameworks, classifications, and terminology. This project serves as a visual bridge between technical security taxonomies and organisational risk management.



## Live View
**Explore the interactive map here: [https://yuriylesyuk.github.io/vuln-cmap/](https://yuriylesyuk.github.io/vuln-cmap/)**

## Key Features
* **Native Scaling:** Fits any screen size while preserving aspect ratio.
* **Deep Dive:** Use the "Open SVG" menu option for native browser zoom and search (Ctrl+F).
* **Interactivity:** Clickable nodes linking to official framework documentation (NIST, MITRE, etc.).
* **Single Source of Truth:** Versioning is embedded directly within the SVG metadata.

## Built With
* [Diagrams.net (Draw.io)](https://app.diagrams.net/) - Diagram authoring.
* GitHub Pages - Hosting and deployment.
* Vanilla JS/CSS - Interactive wrapper.
